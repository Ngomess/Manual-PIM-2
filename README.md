# 📚 Manual do Sistema Acadêmico - PIM 2

Manual interativo desenvolvido para auxiliar usuários na utilização do **Sistema Acadêmico** criado para o Projeto Integrado Multidisciplinar (PIM 2).

O objetivo é apresentar de forma simples e intuitiva todas as funcionalidades do sistema, permitindo que administradores, professores e alunos compreendam rapidamente como utilizá-lo.

---

## 📖 Funcionalidades

- 🏠 Página inicial de apresentação
- 🔐 Manual da tela de Login
- 👨‍💼 Área do Administrador
- 👨‍🏫 Área do Professor
- 🎓 Área do Aluno
- 🔍 Explicação da Busca Inteligente
- ❓ Perguntas Frequentes (FAQ)
- 🔎 Pesquisa de conteúdo
- ✨ Animações durante a navegação
- 📱 Layout responsivo

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 📸 Demonstração

O manual apresenta capturas de tela das principais funcionalidades do Sistema Acadêmico, facilitando o aprendizado e a navegação dos usuários.

---

## 🎯 Objetivo

Este projeto foi desenvolvido como material de apoio ao Sistema Acadêmico criado para o PIM 2, oferecendo uma documentação clara, organizada e de fácil compreensão para todos os perfis de usuários.
